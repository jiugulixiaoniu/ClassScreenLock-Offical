版本状态说明​

  • 正式稳定版 ✅：经过充分测试，性能稳定，适合生产环境使用
  • 测试版 🔄：包含最新功能，可能存在未知问题，适合尝鲜体验
  • 安全补丁更新 🔒：修复关键安全问题，建议及时升级
  • 功能更新 ✨：新增功能或优化现有功能

下载与安装​

系统要求​

  • 操作系统: Windows 10/11 (64位)
  • 内存: 至少 300MB RAM
  • 存储空间: 至少 1500MB 可用空间

安装步骤​

  1. 点击上方"直接下载"链接获取安装包
  2. 运行安装程序，按提示完成安装
  3. 首次启动可能会要求管理员权限
  4. 根据向导完成基础配置

升级说明​

  • 测试版之间可以直接覆盖安装
  • 从旧版本升级建议先备份配置文件

问题反馈​

如果在使用推荐版本时遇到问题：

  1. 检查已知问题: 查看发行说明
  2. 提交反馈: 创建Issue
  3. 紧急问题: 邮件联系 Lavachicken2512@outlook.com

💡 提示: 建议订阅项目Releases页面，及时获取版本更新通知。

📢 重要提醒: 测试版仅供体验，请勿用于重要工作环境。正式版发布前，数据兼容性可能发生变化。

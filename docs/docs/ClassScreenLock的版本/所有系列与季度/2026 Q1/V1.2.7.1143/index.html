V1.2.7.1143

版本信息​

  • 版本类型： 正式版
  • 发布日期： 2026-01-24
  • 版本号： V1.2.7.1143
  • 版本系列： ClassScreenLock V1.2-2026 Q1
  • 发布计划： 隶属于2026年第Q1季度发布计划

⚠️注意：本版本仍为测试版，仍可能存在未知问题，请注意备份数据和谨慎用于大规模投入使用。

首先，非常感谢各位开发者的辛苦付出！本次更新重点重构了课表系统（时间计划）与安全中心，并修复了多处稳定性问题。

🔧 问题修复​

  1. 锁屏适配 - 优化锁屏界面 UI，适配小屏幕显示
  2. UI 修正 - 修复若干界面显示错误
  3. 定时解锁 - 修复到点不会自动解锁的问题
  4. 按钮层级 - 修复下课按钮的层级显示异常

✨ 功能更新​

  1. 安全中心重构 - 整合锁屏设置迁移至安全中心，并将"密码安全中心"更名为"安全中心"

  2. 初始化页面 - 新增用户协议与开源仓库入口，欢迎专业人士审查协议内容并提出修改建议

  3. 周轮换制表系统 - 全面重构时间计划功能：

    • 支持自定义周数轮换（最多 6 周），需在时间计划页面设置开学日期
    • 每周可独立编辑课程安排，周列表仅显示启用范围内的课表
    • 轮换计算逻辑自动适配实际启用的周数设置

  4. CSES 课表导入 - 新增对 CSES 格式的支持：

    • 兼容 ClassIsland 或 Class-Widgets 导出的 YAML 格式课表
    • 在时间计划页面选择 YAML 导入（而非 JSON）：
    • 导入逻辑自动适配当前启用的周轮换设置

🎨 界面优化​

  1. 课程视图升级 - 七天页签统一为课程卡片模板，布局更清晰直观：

🔧 问题修复​

安全漏洞修复​

  1. 修复安全中心登录验证失败问题
    在安全中心模块中，当管理员创建其他用户账户后，再次使用该账户登录时偶现“密码或用户名错误”提示。现已修正用户凭证验证逻辑，确保新建账户可正常登录。

  2. 修复更新后初始化页面残留问题
    通过直接替换 ClassScreenLock.exe 可执行文件的方式进行版本更新后，首次启动软件时错误地显示初始化配置页面（如同首次安装）。现已优化版本检测机制，更新后自动保留原有配置，直接进入主界面。

安装说明​

  1. 下载 ClassScreenLock.zip 压缩包。
  2. 解压至目标目录（建议使用全新文件夹，或覆盖原有版本前做好备份）。
  3. 以管理员身份运行 ClassScreenLock.exe。
  4. 首次启动时，系统将自动迁移旧版本配置（如检测到），无需手动干预。

注： 若您是从 V1.8.30.3049 之前的版本直接替换升级，建议在覆盖前先正常退出旧版本程序，避免文件占用导致更新失败。

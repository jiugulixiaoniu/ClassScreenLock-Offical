ClassScreenLock V1.8.28.2855 更新公告

版本信息​

  • 版本类型： 测试版更新
  • 发布日期： 2026年2月9日
  • 版本系列： ClassScreenLock V1.8-2026 Q1
  • 发布计划： 隶属于2026年第一季度(Q1)发布计划

📢 引言​

我们很高兴发布 ClassScreenLock V1.8.28.2855 版本。本次更新标志着开发进程中的一个重要里程碑，我们在功能、安全性和用户体验上均取得了重大进展。随着核心功能的稳定与完善，ClassScreenLock 即将迈入正式版的发布阶段。​

⚠️ 重要提示​

注意： 本版本仍为测试版，可能包含未知问题或潜在的不稳定性。​

建议： 在升级前备份重要数据，并谨慎将此版本用于大规模生产环境。​

🎯 更新摘要​

本次更新聚焦于自动化、安全增强与用户体验三大核心领域：​

  • 引入强大的自动化功能
  • 扩展安全监控手段（摄像头拍照）
  • 全面优化网络拦截、日志管理及界面交互逻辑
  • 为正式版的发布奠定坚实基础

✨ 主要功能更新​

1. 自动化功能（新增）​

  • 位置： 侧边栏新增"自动化"入口
  • 功能描述： 允许创建基于事件与规则触发的自动化任务流，可监测正在运行的程序与系统时间
  • 触发与操作： 支持配置"当特定事件触发且满足自定义规则"时，执行预设的14种操作（未来将持续增加）
  • 应用场景： 合理配置可实现如自动关机、程序管理、安全响应等高级功能
  • 界面预览：

2. 摄像头拍照功能（新增）​

  • 功能描述： 新增设备摄像头控制与拍照能力，扩展环境监控维度
  • 特色功能： 支持自定义拍摄相关参数
  • 界面预览：

3. 安全日志系统增强​

  • 导出功能： 支持将安全日志导出为 JSON 或 CSV 格式，便于外部分析与存档
  • 可视化优化： 对日志条目进行分类，并以不同颜色区分显示，提升可读性和排查效率
  • 界面预览：

4. 网络拦截机制强化​

  • 简化流程： 移除了原有的"防护模式"选择列表框
  • 高效拦截： 启用网站拦截后直接同时激活 HOSTS 文件屏蔽、软件层拦截与系统防火墙规则
  • 界面预览：

5. 应用管理逻辑加固​

  • 增强应用黑白名单的执行逻辑
  • 进一步确保被限制的程序无法被访问或运行

🔧 问题修复​

修复网络控制被绕过的问题 (#5)​

  • 问题来源： 社区反馈（QQ交流群）
  • 修复内容： 修复特定情况下网络限制可能被绕过的漏洞
  • 解决方案： 将被限制的网站域名直接加入防火墙规则进行拦截
  • 效果： 从根本上增强网络管控的可靠性

🎨 界面与体验优化​

布局调整​

  1. 登录窗口： 将登录状态悬浮窗口移至右上角，避免与侧边栏组件重叠
  2. 侧边栏： 将"设置"和"关于"组件固定置于底部，确保始终可见

信息展示​

  • 在软件主界面右下角及侧边栏底部添加版本号信息
  • 方便用户快速确认当前版本

启动页优化​

  • 将"旋转等待圆圈"替换为带加载状态提示的进度条
  • 使启动过程更加直观

📈 性能与稳定性​

启动优化​

  • 优化启动时的进程初始化与UI加载顺序
  • 确保所有必要组件完全加载完毕后再呈现主界面
  • 提升启动稳定性

速度提升​

  • 相比前一版本，软件启动加载速度平均提升约50%
  • 注：实际效果因设备性能而异

📝 其他更改​

国际化改进​

  • 进一步完善英文语言翻译文件

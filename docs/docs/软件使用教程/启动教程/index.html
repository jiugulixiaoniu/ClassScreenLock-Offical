启动与初始设置教程

下载软件后，请按以下步骤完成首次启动与基础设置。

第一步：解压与启动​

  1. 您下载到的唯一文件为 ZIP 压缩包，请先将其解压。
  2. 进入解压后的文件夹，双击运行 ClassScreenLock.exe 主程序。

第二步：启动引导​

首次运行将自动弹出启动引导窗口，它将指引您完成必要的初始配置。请按照屏幕提示逐步操作。

第三步：关键配置​

当进入到密码种类选择的引导项目时，您需要进行安全验证方式的选择。请仔细阅读并根据自身需求设置：

  • ⚠️ 请首先决定是否需要启用双重认证 (2FA)
  • ✅ 若选择启用 2FA：
    • 请确保您的手机或设备上已安装 Google Authenticator 或同类动态验证码应用。
    • 引导程序将显示二维码供您绑定。
  • 🔑 若选择不启用 2FA：
    • 请直接选择 “仅密码” 选项。
    • 随后点击 “下一步” 继续。

第四步：完成设置​

跟随引导完成所有项目后，点击 “完成” 按钮。当您看到如下主界面时，即表示初始设置已全部完成，可以开始正常使用 ClassScreenLock 的全部功能了。
